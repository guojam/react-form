# React Form

### FormItem

表单项组件，用于布局

当 FormItem 设置了`name`属性且只有一个子组件时，会在内部使用 FormControl 包裹子组件

### FormControl

表单控件容器，用于 formStore 接管表单域子组件双向数据流
